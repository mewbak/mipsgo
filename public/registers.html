Helpful Info

Write your MIPS assembly in the code editor to the left. To run it, press the green play button at the top of the window. If any errors are found, they will be displayed in the console. Use the blue step-forward button to step through your code one line at a time. The play button can be pressed at any point after clicking the step-forward button.

Press the red button at the top of the window to automatically insert a breakpoint above the currenlty selected line. Press play to continue until the next breakpoint or the end of the program or click the step-forward button to step through the remaining code.

MIPSGO has a few pseudo instructions implemented to make debugging a bit easier. Their names, usage, and functionality is as shown:

pbin $rs  # print R[$rs] as binary
phex $rs  # print R[$rs] as hexadecimal
pdec $rs  # print R[$rs] as decimal
break     # insert breakpoint

Currently only a limited subset of MIPS assembly language's operations are available. Those that are currenlty supported are listed below:

add, addi, sub, slt, sll, j, beq, bne, sw, lw, noop
